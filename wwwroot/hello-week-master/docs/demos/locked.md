## Locked

#### Overview
Locks all days in calendar.

#### HTML Structure
```html
    <div class="hello-week"></div>
```

#### Javascript Initialization
```js
    new HelloWeek({
        locked: true
    });
```

#### Properties and Methods
```js
    const calendar = new HelloWeek({
        locked: true
    });
    calendar.setLocked(false);
```
